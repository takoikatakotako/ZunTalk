# ZunTalk ドキュメント

このフォルダには、ZunTalkの法的文書に関する情報が含まれています。

## 公開URL

利用規約とプライバシーポリシーは以下のURLで公開されています：

- **利用規約**: https://takoikatakotako.github.io/projects/zuntalk/terms.html
- **プライバシーポリシー**: https://takoikatakotako.github.io/projects/zuntalk/privacy.html

これらのURLは以下で使用されています：
- iOS アプリのオンボーディング画面（`OnboardingView.swift`）
- App Store Connect の Privacy Policy URL / Terms of Service URL

## App Store 申請時の設定

App Store Connect で以下のURLを登録：

- **Privacy Policy URL**: https://takoikatakotako.github.io/projects/zuntalk/privacy.html
- **Terms of Service URL** (任意): https://takoikatakotako.github.io/projects/zuntalk/terms.html

## 規約を更新する場合

1. GitHub Pages のリポジトリ（takoikatakotako.github.io）で HTML を更新
2. URLは変更しないことを推奨（変更する場合は以下も更新が必要）：
   - iOS アプリの URL（`OnboardingView.swift`）
   - App Store Connect の URL
   - アプリのアップデート申請

## 注意事項

- 規約の実ファイルは別のリポジトリ（takoikatakotako.github.io）で管理されています
- 重要な変更を行った場合は、既存ユーザーへの通知を検討してください
- 必要に応じて弁護士や専門家に相談することをおすすめします
