プライバシーポリシー

最終更新日: 2026年6月6日

takoikatakotako（以下「当方」といいます）は、スマートフォン向けアプリケーションの開発を行っております。本アプリ「ZunTalk」（以下「本アプリ」といいます）をご利用いただくにあたり、以下のプライバシーポリシーをご確認ください。本アプリをご利用いただいた場合、本プライバシーポリシーに同意いただいたものとみなします。

収集した個人情報は、サービスの提供および改善のために使用されます。

1. サポート

本アプリでは、サポートのため、デバイスの種類やOSのバージョンといった情報を取得する場合があります。これらの情報は、不具合が発生した際の原因究明に役立てるためです。

2. データ解析

本アプリでは、アプリの利便性向上のため、アクセス解析やクラッシュログの解析を行う場合があります。これらの解析には、Google が提供する Firebase Analytics および Firebase Crashlytics を利用しています。

収集する場合がある情報は以下のとおりです。

  • アプリの利用状況
  • デバイス情報
  • OSバージョン
  • エラー情報・クラッシュ情報

これらの情報は、サービスの品質向上およびユーザー体験の改善を目的として使用されます。Firebase における情報の取り扱いについては、Google プライバシーポリシーをご確認ください。

3. 広告の配信について

本アプリでは、Google が提供する広告配信サービス「Google AdMob」を利用して広告を表示します。AdMob は、広告の配信および最適化のため、広告識別子（IDFA）、デバイス情報、IPアドレス、おおよその位置情報、広告データ（広告の表示・タップ等の利用状況）、および SDK の診断・性能情報（クラッシュログ、パフォーマンスデータ）などを収集・利用する場合があります。詳細は AdMob のデータ開示 をご確認ください。

iOS の「App のトラッキングの透明性（ATT）」に基づき、本アプリは広告識別子（IDFA）の利用についてユーザーに許可を求めます。トラッキングを許可しない場合でも広告は表示されますが、その場合、Google Mobile Ads SDK は広告識別子（IDFA）を送信せず、IDFA を利用したパーソナライズ（興味・関心に基づく広告の最適化）は行われません。

広告配信における情報の取り扱い、およびパーソナライズ広告のオプトアウトについては、以下をご確認ください。

  • Google による広告での情報の使用: https://policies.google.com/technologies/partner-sites
  • 広告設定（オプトアウト）: https://adssettings.google.com/

4. 外部サービスプロバイダ

本アプリでは、以下の目的のために外部のサービスプロバイダと契約し、個人情報を提供する場合があります。

  • サービスの機能を容易にするため
  • 当方に代わってサービスを提供するため
  • サービスに関連するサービスを実行するため
  • サービスがどのように使用されているかの分析を支援するため

AIサービスの利用について

本アプリは、AI応答を生成するため、ユーザーが入力したメッセージを外部のAIサービスプロバイダ（OpenAI, L.L.C. 等）に送信します。送信された情報は、AI応答の生成のために使用され、各サービスプロバイダのプライバシーポリシーに従って取り扱われます。

  • OpenAI API
    • プライバシーポリシー: https://openai.com/policies/privacy-policy

その他の外部サービスプロバイダ

  • Google（Firebase / AdMob）
    • プライバシーポリシー: https://policies.google.com/privacy

ユーザーは、本アプリを使用することで、外部サービスプロバイダによる情報の収集および使用に同意したものとみなされます。
